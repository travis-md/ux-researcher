# Context Onboarding — Build Your Setup in Chunks

Don't try to do this all at once. Each module takes 10 minutes and produces something immediately usable.

## The modules

| Module | What it builds | Time | Do it when |
|---|---|---|---|
| **1 — The Basics** | Role, team, tools | 10 min | Today — start here |
| **2 — How You Work** | Research style, process, patterns | 10 min | When you have a free 10 |
| **3 — Your Stakeholders** | Who you work with and for | 10 min | Before your next readout |
| **4 — AI Working Style** | How you want Claude to work with you | 10 min | After using it for a week |
| **5 — Current Project** | Deep context on what you're working on now | 10 min | Start of each new project |

## How it works

Each module is a prompt you paste into Claude. Claude interviews you — asks questions, you answer in plain language. At the end it writes a context file section you can save.

Run Module 1 today. Everything after that is optional and additive.

## Your file builds over time

```
context-v1.0.md    ← Module 1 only. Enough to get started.
context-v1.1.md    ← Added Module 2. Claude knows how you work.
context-v1.2.md    ← Added Module 3. Claude knows your stakeholders.
context-v2.0.md    ← Full picture. All 5 modules.
```

Each version works. You just get more out of it as it grows.
